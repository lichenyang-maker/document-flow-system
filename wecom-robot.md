# 企业微信 / 微信推送接入指南

## 三种推送方式

| 方式 | 用途 | 需要什么 | 配置位置 |
|------|------|---------|---------|
| **企业微信机器人** | 推送消息到企业微信群 | 群机器人 webhook URL | 系统设置 → 微信配置 |
| **Server酱** | 推送消息到个人微信 | Server酱 SendKey | 同上 |
| **PushPlus** | 推送消息到个人微信 | PushPlus Token | 同上 |

---

## 一、企业微信机器人（推荐）

### 创建机器人
1. 打开企业微信群聊
2. 点击群设置 → 群机器人 → 添加机器人
3. 给机器人起名 → 复制 webhook URL
4. webhook URL 格式：`https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=xxx`

### 在系统里配置
管理员登录 → 用户管理 → 微信配置 → 选择「企业微信」→ 粘贴 webhook URL → 保存

---

## 二、Server酱（推送到个人微信）

1. 访问 https://sct.ftqq.com/ ← 用 GitHub 登录
2. 复制你的 SendKey
3. 系统里选择「Server酱」→ 粘贴 SendKey

---

## 三、测试推送

配置保存后，点击「测试推送」按钮，系统会发一条测试消息到你配置的渠道。

---

## 四、系统会自动推送的场景

| 场景 | 推送到 |
|------|--------|
| 有新的请假申请 | 微信 |
| 请假审批通过/驳回 | 申请人微信 |
| 每日简报（每天9:00） | 配置的群/个人 |
| 待审批提醒（每30分钟） | 审批人群 |
